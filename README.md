AfriTalent

Présentation

AfriTalent est une plateforme web moderne qui met en relation les entreprises avec les meilleurs freelances africains. Le projet a été développé dans le cadre d’un apprentissage en développement web en utilisant HTML, CSS, Bootstrap et JavaScript.

L’objectif est de faciliter la recherche de talents qualifiés dans différents domaines tels que le développement web, le design graphique, la rédaction web, le marketing digital et bien d’autres.
